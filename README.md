ERP Management System – Project Description

The ERP Management System is a Python-based desktop application developed to streamline and manage customer and business records efficiently. The system integrates a MySQL database with Python to perform complete CRUD (Create, Read, Update, Delete) operations through a simple and user-friendly interface. It allows users to add, view, update, search, and delete records while ensuring data is stored securely in a centralized database.

The project demonstrates practical implementation of database management, SQL queries, Python programming, and modular application development. It is designed to improve data organization, reduce manual record-keeping, and provide quick access to business information.

Add, View, Update, and Delete Operations
